泛型(Generic)

发表于2022-11-20|更新于2023-03-20|Programming Language
|阅读量:

泛型(Generic)

通用数据类型

  • 对数据元素进行约束，编译时就能看到是否有类型错误的问题；

  • 避免不必要的类型转换，提高编译效率(并不能提高运行效率)；

泛型类

自定义泛型类

class 类名<T,R>{

}

  • 在实例化类的时候指定其元素运行类型<E, T, ….> 可以有多个不同类型， 一般用单个大写字母表示数据类型；

  • 泛型只能是引用类型，不能是基本类型

  • 使用泛型的数组，不能初始化

  • 静态方法中不能实用泛型

  • 泛型的类型，是在创建对象时确定的

泛型接口

语法

interface class_name<T, R>{

}

  • 接口中成员类型，不能使用泛型
  • 泛型接口的类型在继承接口或者实现接口时确定

继承泛型接口

实现泛型接口

  • 没有指定类型，默认为Object

泛型方法

语法： 修饰符 <T …>返回值 方法名(参数列表){ }

  • 泛型方法，可以定义在泛型类，还可以普通类中
  • 当泛型方法被调用时，类型会确定
  • 方法中没有使用<>定义泛型， 就不是泛型方法

泛型通配符

用于约束泛型

  • 泛型类型不具备继承特性
  • 表示任意泛型类型都可以接收
  • 上界通配符
  • 下界通配符

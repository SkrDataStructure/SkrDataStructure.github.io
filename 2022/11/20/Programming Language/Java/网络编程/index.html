网络编程

网络通信协议：

OSI模型：

  • 1）应用层
  • 2）表示层
  • 3）会话层
  • 4）传输层 (TCP, UDP..)
  • 5）网络层 (IP, ICMP..)
  • 6）数据链路层(Link)
  • 7）物理层 (Link)

TPC/IP模型（简化的OSI模型）

  • 1）应用层(应用数据)
  • 2）传输层(TCP)
  • 3）网络层(IP)
  • 4）数据链路+物理层(Link-+)

InetAddress

用户获取Address对象

  • 常用方法

Socket

Socket使用TCP/IP协议，是基于客户端-服务端的网络通信

通信通道，两边的端点采用Socket

Socket允许程序把网络链接当成一个流，数据在两个Socket间通过IO流传输数据

一般主动发起通信的应用属于客户端，等待监听对方请求信息的称为服务端
